MCloud SaaS

A production‑ready Minecraft Bedrock hosting platform

Overview Setup Architecture Contribute

Overview

MCloud SaaS provides one‑click provisioning of Minecraft Bedrock servers, integrated payment via Midtrans, and an admin dashboard with real‑time console access. The platform runs as four Docker‑containerised micro‑services orchestrated by PM2.

Setup & Deployment

Follow the Production Setup Guide for a step‑by‑step deployment on a VPS or cloud server.

Architecture

Four services (Gateway, Auth, Payment, Provisioning) exposed behind an Nginx reverse‑proxy with SSL.
