SYSTEM CAPABILITIES

Why MCloud SaaS Stands Out

Engineered from the ground up for high concurrency, robust security, and seamless developer experience.

⚡

Ultra-Fast Provisioning

Powered by native Docker Engine socket binding (`unix:///var/run/docker.sock`). New game nodes are provisioned with volume isolation and custom port mapping in under 3 seconds.

🔄

PM2 Cluster Self-Healing

Orchestrated by a custom PM2 ecosystem configuration. All 4 microservices feature automatic memory overload restart, unified error logging, and system startup persistence.

🛡️

Enterprise Security

Equipped with Let's Encrypt SSL/TLS reverse proxying, UFW firewall rules protecting internal service ports (3000-3003), and anti-DDoS game port allocation (25500:25600).

📊

Interactive Admin Suite

A comprehensive React 19 control panel featuring live polling of CPU/RAM stats, structured Support Ticket management, historical release Changelogs, and system FAQs.

💾

Automated Maintenance

Includes automated nightly Linux cron jobs that compress and archive SQLite databases while pruning dangling Docker images to optimize NVMe SSD storage.

🧩

Monorepo Architecture

Clean separation of concerns with dedicated apps for Gateway, Auth, Payment, and Provisioning, sharing common validation types and Drizzle ORM database schemas.

TECHNICAL DESIGN

Microservices & Infrastructure Flow

How our Nginx Reverse Proxy, PM2 Cluster, and Docker Engine collaborate in production.

🌐 API Gateway (Port 3000)

The central entry point handling rate limiting, request validation, and WebSocket upgrades for live terminal streaming.

🔐 Auth Service (Port 3001)

Manages stateless JWT authentication, secure bcrypt password hashing, and user role authorization (Admin vs Client).

💳 Payment Service (Port 3002)

Directly interfaces with Midtrans Snap API (Production & Sandbox) and processes real-time HTTP notification webhooks.

🐳 Provisioning Service (Port 3003)

Executes commands against the local Docker Daemon to create, start, stop, restart, and delete Minecraft server containers.
