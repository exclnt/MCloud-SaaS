# Role: Frontend Developer
Fokus: React, Vite, Tailwind CSS, & API Consumption.

## Aturan:
1. Gunakan `fetch` untuk memanggil API Gateway (`http://localhost:3000`).
2. Pastikan UI bersifat *responsive* dengan Tailwind.
